CampusCraze

The Problem:

Students often receive college symposium and event posters after registration deadlines or event dates have already passed, leading to missed opportunities and confusion.

The Solution:

Built a centralized mobile platform where educational event and symposium information can be shared in real time, ensuring students receive accurate and timely updates without delays.

Key Features:

  • Centralized posting of college event, education events and symposium details
  • Real-time access to event information for students
  • Simple and student-friendly interface for easy navigation

Tech Used:

Flutter (Dart), Firebase Firestore, Firebase Authentication, SQLite

My Contribution:

Identified the problem from personal experience, designed the solution, and independently developed the complete application using Flutter and Firebase, focusing on reliability, clarity, and real-time data handling.

Travel Duty Management

The Problem:

Managing travel duty schedules manually often leads to confusion, missed updates, and difficulty in tracking ongoing and completed duties, especially when handling multiple customers and dates.

The Solution:

Developed a mobile application that digitally organizes travel duty details, separates ongoing and completed duties, highlights priority schedules, and allows quick sharing of information with customers.

Key Features:

  • Ongoing and Finished duty tabs for clear task separation
  • Priority highlighting for today's and tomorrow's duties
  • Date-based sorting in descending order for quick access
  • Edit, delete, and search functionality for duty records
  • Duty sharing feature for customer communication
  • Secure Gmail-based user authentication

Tech Used:

Flutter (Dart), Firebase Authentication, Firebase Firestore, Firebase Remote Config

My Contribution:

Designed and developed the complete application independently, implemented Firebase services, and applied priority-based logic and sorting to improve usability and efficiency.
