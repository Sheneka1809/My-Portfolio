BST-Based Dictionary - Learning Experiment

Objective:

To explore how a Binary Search Tree can be used to implement a simple dictionary and to understand its behavior in comparison with Trie-based approaches.

Approach:

Implemented a small dictionary with a limited set of words and meanings using a Binary Search Tree. Although Trie is the standard structure for dictionaries, BST was intentionally chosen to experiment with alternative data structure usage and internal operations.

Key Learnings:

  • Binary Search Trees can be used for basic dictionary lookups but are not optimal for prefix-based searches
  • Trie structures are better suited for large-scale dictionary implementations
  • Data structure selection should be based on problem requirements rather than conventions

Banking Application using Tkinter - Learning Project

Objective:

To understand the basics of application development and user interaction by building a simple banking application using Python's Tkinter library.

Approach:

Developed a basic desktop banking application using Tkinter during the initial learning phase. The app allows users to create an account with a username and PIN, log in using those credentials, and perform simulated banking operations such as deposit, withdrawal, and balance checking. No database was used, as the focus was on application logic and UI flow.

Key Learnings:

  • Gained early experience in building GUI-based applications
  • Understood event handling and control flow in desktop applications
  • Learned how application logic works before integrating databases
  • Built a strong foundation that later helped transition smoothly to Flutter
